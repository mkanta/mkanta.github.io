1. Kinematics

1.1. Projectiles

Projectile motion is described by the equation \[\vec{d}(t)=\vec{v}_it+\frac{1}{2}\vec{a}t^2\] with 2-dimensional vectors \(\vec{d}=(d_x,d_y)\) describing displacement from an initial position, initial velocity \(\vec{v}_i=(v_x,v_y)\) and acceleration \(\vec{a}=(a_x,a_y)\). If the coordinate system is chosen to have its origin at the beginning of the motion then \(\vec{d}(t)\) can be interpreted as the position of the projectile at time \(t\). For projectile motion, only the vertical component experiences acceleration caused by gravity, hence \[\vec{a}=(0,-9.81).\] The initial velocity \(\vec{v}\) given by an initial speed \(s\) at an angle \(\alpha\) with respect to the horizontal is \[\vec{v}=(s\cos\alpha,s\sin\alpha).\] In particular, an initial velocity in the horizontal direction is characterised by an angle \(\alpha=0\), reulting in a 2-dimensional vector \(\vec{v}_i=(s,0)\).

  1. An object is launched at \(15.0\frac{\text{m}}{\text{s}}\) at an angle of \(30.0^\circ\) to the horizontal. Determine its range and maximum height.

    Click to see the answer, hints or generate more questions of this type.

    Range:\(19.9\)m, Height:\(2.87\)m

    1. The initial velocity is \(\vec{v}=(15.0\cos 30.0,15.0\sin 30.0)\) and the acceleration
    2. is \(\vec{a}=(0,-9.81)\). This motion is described by
    3. the equation \[(d_x,d_y)=(15.0\cos 30.0 ,15.0\sin 30.0 )t+\frac{1}{2}(0,-9.81)t^2,\]
    4. which has horizontal component \(d_x=15.0\cos 30.0 t\) and vertical component
    5. \(d_y=15.0\sin 30.0 t-\frac{1}{2}9.81t^2\).
    6. Since the range is to be determined, the time \(t\) has to be found when the object hits the ground, ie when the vertical displacement from the ground is \(d_y=0\).
    7. From the vertical component of the motion, this puts the requirement \[0=15.0\sin 30.0 t-\frac{1}{2}9.81t^2=t(15.0\sin 30.0 -\frac{1}{2}9.81t)\] on \(t\).
    8. This is satisfied when \(t=0\) or \[15.0\sin 30.0 -\frac{1}{2}9.81t=0,\]
    9. with the latter \(t\) being of interest because that represents the time when the object lands.
    10. Solving for \(t\), this becomes \[t=\frac{30.0\sin 30.0 }{9.81}.\]
    11. To find the range, this has to be inserted back into the horizontal component, which becomes \[d_x=15.0\cos 30.0 \frac{30.0\sin 30.0 }{9.81}=\frac{15.0\times 30.0\sin 30.0 \cos 30.0 }{9.81}=19.9,\] so the range is \(19.9\)m.
    12. For the height, \[0=\sin^230.0 15.0^2-2\times9.81\times d_y\] needs to be computed,
    13. which leads to \[d_y=\frac{15.0^2\sin^230.0}{2\times 9.81}=2.87,\] a height of \(2.87\)m.
